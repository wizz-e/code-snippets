# Code Snippets

This directory contains organized code snippets for various programming languages.

## Structure

Each subdirectory represents a programming language:

- **python/** - Python code snippets
- **javascript/** - JavaScript/TypeScript snippets
- **java/** - Java code snippets
- **cpp/** - C/C++ code snippets
- **go/** - Go code snippets
- **rust/** - Rust code snippets
- **shell/** - Shell/Bash script snippets
- **sql/** - SQL query snippets

## Naming Convention

Use descriptive names for your snippet files:
- `function_name.ext` for single functions
- `topic_name.ext` for related snippets
- Include comments explaining what the snippet does
